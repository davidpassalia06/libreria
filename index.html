Sincronizzazione Firebase ☁

Sincronizza tra dispositivi tramite Firebase Realtime Database — gratuito, senza server.

Come configurare — 5 minuti, gratis

  1. Vai su console.firebase.google.com
  2. Crea un nuovo progetto (piano gratuito Spark)
  3. Vai su Realtime Database → Crea database → Modalità test
  4. Copia l'URL del database (https://nome.firebaseio.com)
  5. Incollalo qui sotto, salva, poi usa i pulsanti per sincronizzare
